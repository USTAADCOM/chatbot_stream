# ChatBot With Stream
This is a chatbot with real time stream like ChatGpt.

## Setup
  
  clone project with Python 3.10.10
  ```code
  git clone https://github.com/USTAADCOM/chatbot_stream.git
  cd chatbot_stream
  pip install -r requirements.txt
  ```
## Project Structure

```bash
chatbot_stream
   │   app.py
   │   requirements.t
   │
   ├───static
   │       chat.js
   │       style.css
   │
   └───templates
           chat.html
```

## Run Server  
```code
python app.py
```

## Streaming ChatBot 
```code
http://127.0.0.1:5000/chat
```
